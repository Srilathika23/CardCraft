# CardCraft

CardCraft is a web-based application that enables users to create and personalize digital greeting cards for various occasions. The platform provides an intuitive interface for customizing card designs through editable text, fonts, colors, and layouts, allowing users to create unique and visually appealing greeting cards.

## Live Demo

https://cardcraft-wheat.vercel.app/

## Features

- Create personalized greeting cards for different occasions
- Customize text, fonts, colors, and layouts
- Responsive and user-friendly interface
- Real-time card preview
- Clean and intuitive design

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Srilathika23/CardCraft.git
```

2. Navigate to the project directory:

```bash
cd CardCraft
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
